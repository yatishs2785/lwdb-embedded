# lwdb-embedded
Lightweight, Embedded‑Friendly SQL‑Like Database Engine (Modern C++)
lwdb-embedded is a lightweight, modular, SQL‑like embedded database library written in modern C++, designed for embedded systems, edge devices, and resource‑constrained environments.
It provides a minimal relational database core with a clean API, adapter‑based integration, and a plugin‑ready architecture — without any dependency on SQLite.
